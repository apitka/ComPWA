Detailed Description

Test-Application to check c++11 support.

This tiny application test some features of the new c++11 standard. One can compile and run this to check the c++11 support of the used compiler/system. The features tested are: shared_pointer, raw strings, auto type's, lamdas, threading and regular expressions.
