Select random sub sample of data sets.

A hit&miss procedure is applied to the first sample to select a random set of events. In case an event of the first sample is added to the output sample, an event from the second sample is also added to the corresponding output sample. This function can be used for two sample that are in sync. E.g. a sample with reconstructed values and a sample with the corresponding true values We expect that
