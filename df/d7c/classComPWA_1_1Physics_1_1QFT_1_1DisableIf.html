Detailed Description

template<bool _condition, typename _Tp = void>
class ComPWA::Physics::QFT::DisableIf< _condition, _Tp >

Used to disable overloaded template function given a condition.

Author
Mike Williams

DisableIf<condition,T> defines the type Type only if condition is false. Thus, this template can be used to disable an overloaded template function when the condition is true.
