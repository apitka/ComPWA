BOOST_AUTO_TEST_CASE ( HelicityAngleTest  )

Test application for the calculation of the helicity angle.

As example the decay D0->KsK-K+ is used. Some test may be specific to this decay but if all test are passed we can be sure that helicity angles are calculated correctly.

A note on numerical precision: We compare all valued using float precision only since otherwise variables calculated using different methods are not completely equal on double precision. Every not an then even with float precision the test fails.
