Detailed Description

HelicityKinematics class.

Implementation of the ComPWA::Kinematics interface for amplitude models using the helicity formalism.

Create HelicityKinematics from inital and final state particle lists.

The lists contain the pid of initial and final state. The position of a particle in initial or final state list is used later on for identification.

Definition at line 33 of file HelicityKinematics.cpp.

ComPWA::Physics::HelicityFormalism::HelicityKinematics::HelicityKinematics ( boost::property_tree::ptree  pt)

Create HelicityKinematics from a boost::property_tree.

The tree is expected to contain something like:

<HelicityKinematics>
<PhspVolume>1.45</PhspVolume>
<InitialState>
<Particle Name='jpsi' Id='0'/>
</InitialState>
<FinalState>
<Particle Name='pi0' Id='1'/>
<Particle Name='gamma' Id='0'/>
<Particle Name='pi0' Id='2'/>
</FinalState>
</HelicityKinematics>

The Id is the position of the particle in input data.
